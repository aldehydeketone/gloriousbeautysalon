# Glorious Beauty Salon Website

A modern, minimal, and premium website for a ladies-only beauty salon.

## Features
- **Premium Design**: Clean aesthetics with glassmorphism effects and soft color palette.
- **Responsive**: Fully responsive design with a mobile-friendly navigation menu.
- **Animations**: Smooth scroll effects and entry animations.
- **Tech Stack**: Built with Vite and Vanilla JavaScript for high performance.

## How to Run

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Open in Browser**:
    Visit the URL shown in the terminal (usually `http://localhost:5173/`).

## Customization
- **Colors**: Edit CSS variables in `src/style.css` (e.g., `--accent`, `--secondary`).
- **Content**: Update `index.html` with real text and images.
